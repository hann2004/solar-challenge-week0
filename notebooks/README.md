# Notebooks Directory

This directory contains Jupyter notebooks for solar data analysis:

## Notebook Files:

- `benin_eda.ipynb` - Exploratory Data Analysis for Benin solar data
- `sierra_leone_eda.ipynb` - EDA for Sierra Leone solar data  
- `togo_eda.ipynb` - EDA for Togo solar data
- `compare_countries.ipynb` - Cross-country comparison and statistical analysis

## Usage:

Run these notebooks in order to reproduce the analysis:
1. Start with country-specific EDA notebooks
2. Run the comparison notebook for cross-country insights

All notebooks use the cleaned CSV files from the `data/` directory.
